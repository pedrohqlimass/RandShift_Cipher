RandShiftCipher

Projeto desenvolvido para a matéria de Atividades Práticas
Supervisionadas (APS) do curso de Ciências da Computação – 2º semestre.

O objetivo do RandShiftCipher é aplicar conceitos de lógica de
programação, manipulação de strings, criptografia simples e organização
modular de código, criando um criptografador baseado em um deslocamento
randômico (similar ao método Caesar Cipher, porém com chave dinâmica).

Descrição do Projeto

O RandShiftCipher é um programa que:

-   Gera um deslocamento aleatório para criptografia.
-   Aplica um algoritmo simples de criptografia de substituição.
-   Permite descriptografar o texto usando o mesmo deslocamento.
-   Faz inversão final da mensagem para aumentar a obfuscação.
-   Salva o resultado em um arquivo .txt.
-   Exibe uma logo ASCII personalizada no início da execução.
-   O usuário pode realizar várias operações no mesmo ciclo do programa.

Tecnologias Utilizadas

Python 3

Manipulação de: - Strings - Funções - Entrada/saída - Arquivos - Lógica
de controle

Nenhuma biblioteca externa foi utilizada, apenas recursos nativos da
linguagem.
